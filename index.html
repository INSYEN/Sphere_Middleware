Sphere middleware

A python-based middleware for DTNMP control.

Download .zip Download .tar.gz View on GitHub

Sphere_middleware is the (admittedly uncreative) name for the python-based middleware which composes the core of our DTNMP commanding system.

What Is This?

This package provides an end-to-end solution for DTN network management. It is comprised of three components: 1 dtnmp_commanding - A heavily-modified version of the DTNMP reference implementation from ION, enhanced with RPC, Datalists, and new commanding capabilities for ION. This communicates via TCP with... 2 sphere_middleware - A python middleware which handles all higher-level control of the DTNMP nodes, as well as reporting, archival, simple routing algorithms, etc. This software has an HTTP interface, and communicates with... 3 SphereWebFrontend - An HTML/Javascript based web application, which allows simple control of DTN networks.
